# P-gina-web-do-zero-primeira-intera-o
Benefícios da fruta Banana para a saúde
Atue como um Desenvolvedor Front-end Sênior. Crie o código para um site responsivo e moderno sobre "Os Benefícios da Banana para a Saúde Humana e suas Vitaminas". 

O site deve ser construído utilizando apenas HTML5 estruturado e semântico, CSS3 moderno (com Flexbox/Grid e variáveis para cores) e JavaScript Vanilla (nativo).

### Diretrizes de Design e Identidade Visual (CSS):
- Use uma paleta de cores limpa e temática: Amarelo suave (#FFDE4D), Verde-folha (#4CAF50), Branco e Cinza escuro para os textos.
- Design responsivo (mobile-first) que funcione perfeitamente em celulares e computadores.
- Efeitos suaves de transição (hover) nos botões e cards.

### Estrutura do Site (HTML) e Funcionalidades (JavaScript):
1. HEADER: Menu de navegação simples e um logotipo fictício.
2. HERO SECTION: Título de impacto, subtítulo motivador e uma imagem ilustrativa (pode usar um placeholder do Unsplash).
3. SEÇÃO DE VITAMINAS: Cards interativos que mostram os nutrientes (Potássio, Vitamina B6, Vitamina C, Triptofano). Use JavaScript para que, ao clicar em um card, um modal ou um texto extra se abra revelando os detalhes do benefício de cada nutriente.
4. CALCULADORA NUTRICIONAL (JS): Uma ferramenta simples onde o usuário digita a quantidade de bananas (ex: 1, 2, 3) e o JavaScript calcula e exibe na tela a estimativa de Calorias (aproximadamente 89kcal por unidade) e Potássio (358mg por unidade).
5. SEÇÃO DE RECEITAS: Uma lista de 3 receitas rápidas com abas ou um filtro simples em JavaScript (Ex: "Café da Manhã", "Pós-Treino").
6. FOOTER: Direitos autorais e links sociais fictícios.

Por favor, forneça os códigos separados em blocos: index.html, style.css e script.js. Certifique-se de que o código seja limpo, bem comentado e pronto para execução.
